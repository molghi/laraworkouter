# Workout Companion

A simple web app to plan and track workouts. Built with Laravel and Tailwind CSS.

## Features

-   Add and time workouts with exercise/rest durations
-   Save completed sessions automatically
-   View workout history with filters
-   Calendar highlighting active days
-   Basic stats: total time, frequency, averages
-   User authentication (signup/login)
-   Clean, responsive interface built with Tailwind CSS
